### **Aim**
> Installation and exploration of MySQL.

### **Requirements**
- **Hardware:** Personal Computer/Laptop
- **Software:** Windows/Linux OS, MySQL Installer (Community Server)

### **Procedure**
- Download the MySQL Installer from the official website.
- Run the installer and choose the setup type (e.g., "Developer Default" or "Server only").
- Configure the server (Port 3306 is default).
- Set the Root Password (required for login).
- Complete the installation and start the MySQL server.
