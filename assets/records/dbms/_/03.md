### **Aim**
> To write SQL queries for table creation and modification

### **Queries**
- Create a table with constraints
```sql
-- Create database (if not selected)
USE LabRecordDB;

-- Create table with constraints
CREATE TABLE Employee (
    EmpID INT PRIMARY KEY,              -- Unique ID for each employee
    FirstName VARCHAR(50) NOT NULL,     -- Cannot be empty
    LastName VARCHAR(50),
    Email VARCHAR(100) UNIQUE,          -- No duplicate emails allowed
    Salary DECIMAL(10, 2) CHECK (Salary > 0), -- Validation check
    HireDate DATE
);

-- Verify the structure
DESC Employee;
```

- Add a new column to the table
```sql
ALTER TABLE Employee
ADD Department VARCHAR(30);

-- Verify the change
DESC Employee;
```

- Modify an existing column
```sql
ALTER TABLE Employee
MODIFY LastName VARCHAR(100);

-- Verify the change
DESC Employee;
```

- Rename a column
```sql
ALTER TABLE Employee
RENAME COLUMN HireDate TO DateOfJoining;

-- Verify the change
DESC Employee;
```

- Drop a column
```sql
ALTER TABLE Employee
DROP COLUMN Email;

-- Verify the change
DESC Employee;
```

- Rename the table
```sql
RENAME TABLE Employee TO Staff;

-- Verify the table list
SHOW TABLES;
```

### **Conclusion**
> Successfully utilized DDL commands to define and manipulate the database schema.
